Task 2: Toggle Visibility Feature 
Objective: Implement conditional rendering using state. 
Requirements: 
1. Create a button with text: 
○ “Show Details” (initially) 
2. When clicked: 
○ Display a paragraph with some details. 
○ Button text should change to “Hide Details”. 
3. When clicked again: 
○ Hide the paragraph. 
○ Button text should switch back to “Show Details”. 
4. Use React’s useState hook.

Task 3: Simple User Dashboard 
Objective: Create a small dynamic dashboard with reusable components and state 
updates. 
Requirements: 
1. Create a reusable component named UserCard. 
2. Each UserCard should display: 
○ User name 
○ User role 
○ “Promote” button 
3. Maintain a list of 3 users in the parent component using state. 
4. When “Promote” is clicked: 
○ Update that specific user’s role to “Senior”. 
5. Ensure proper re-rendering without directly mutating state. 
6. Use unique keys while rendering list

Task 1: Reusable Product Card Component 
Objective: Build a reusable ProductCard component to display product information and handle 
user interaction. 
Requirements: 
1. Create a functional component named ProductCard. 
2. It should accept the following props: 
○ product name  
○ price 
○ function to handle add-to-cart action 
3. Display: 
○ Product Name 
○ Price 
○ “Add to Cart” button 
4. When the button is clicked: 
○ Log the message: 
"Added {productName} to cart" in the console. 
5. Use the ProductCard component multiple times in the parent component with different 
products.
